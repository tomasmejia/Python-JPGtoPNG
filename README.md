## Installation

This project depends on PIP/Pillow. 
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Pillow.

```bash
pip3 install Pillow
```

## Usage

```python JPGtoPNG.py [image_folder] [output_folder]```
